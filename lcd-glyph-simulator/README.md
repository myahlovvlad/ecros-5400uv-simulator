# LCD 128×64 Glyph Simulator

Готовый web‑проект для рендера пиксельных глифов на симуляторе монохромного LCD‑дисплея 128×64.

Что входит:
- `index.html` — основное приложение.
- `styles.css` — стили.
- `glyphs-data.js` — скомпилированная база глифов.
- `app.js` — логика симулятора, браузера глифов и экспорта framebuffer.
- `glyphs.compiled.json` — machine‑readable представление спецификации.
- `lcd-glyph-simulator-standalone.html` — автономная версия в одном файле.

Возможности:
- выбор варианта шрифта 5 px / 6 px / 8 px;
- рендер текста на LCD 128×64;
- управление масштабом, межсимвольным и межстрочным интервалом;
- инверсия, сетка, автоматический перенос;
- браузер глифов с предпросмотром;
- диагностика несоответствий в исходной спецификации;
- экспорт текущего framebuffer в `uint8_t lcd_buffer[1024]`.

Запуск:
1. Откройте `lcd-glyph-simulator-standalone.html` прямо в браузере.
2. Либо откройте `index.html` вместе с соседними файлами.

Примечание:
Исходная спецификация содержит несколько внутренних несоответствий между заявленными размерами и фактическими матрицами. Они не теряются, а выводятся в блоке диагностики.
