# Статус реализации

## Что реализовано

### Core

- модель измерений и расчёта оптической плотности
- стартовое состояние прибора
- градуировка, кинетика и файловые операции
- USB preview для экспортируемых данных

### UI

- web-макет панели управления
- bitmap LCD 128x64 с пользовательским шрифтом
- графики фотометрии, кинетики и градуировки
- боковые панели статуса, CLI и навигации
- редактор подписей панели
- редактор строк LCD

### Техническая часть

- Vite-сборка
- Vitest-тесты
- Windows launcher для локального запуска
- `main` и тег `v2.0.0` опубликованы в GitHub

## Проверено

Актуальная проверка проекта:

- `npm run build`
- `npm test -- --run`
