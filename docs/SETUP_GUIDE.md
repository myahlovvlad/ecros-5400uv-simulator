# Руководство по запуску и установке

## Быстрый старт

### 1. Автоматическая установка (Windows)

Запустите файл `init.bat` двойным кликом или через командную строку:

```cmd
init.bat
```

Скрипт автоматически:
- Проверит наличие Node.js и npm
- Создаст структуру папок
- Скопирует файлы в папку docs/
- Установит все зависимости
- Установит jsdom для тестов

### 2. Ручная установка

#### Требования
- Node.js >= 18.0.0
- npm >= 9.0.0

#### Шаг 1: Установка зависимостей

```bash
npm install
npm install jsdom --save-dev
```

#### Шаг 2: Запуск разработки

```bash
npm run dev
```

Приложение откроется в браузере по адресу `http://localhost:3000`

#### Шаг 3: Сборка для продакшена

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`

## Устранение проблем

### Ошибка "Node.js не найден"

Установите Node.js с официального сайта: https://nodejs.org/

### Ошибка при установке зависимостей

Очистите кэш npm и попробуйте снова:

```bash
npm cache clean --force
npm install
```

### Ошибка при запуске тестов

Убедитесь, что jsdom установлен:

```bash
npm install jsdom --save-dev
```

### Ошибка сборки

Удалите папку `node_modules` и `package-lock.json`, затем:

```bash
npm install
npm run build
```

## Проверка работоспособности

После установки проверьте:

1. **Сборка:**
   ```bash
   npm run build
   ```
   Должно завершиться успешно с сообщением "✓ built in XXXms"

2. **Тесты:**
   ```bash
   npm test -- --run
   ```
   Должны запуститься тесты (могут быть ошибки в тестах)

3. **Разработка:**
   ```bash
   npm run dev
   ```
   Откроется браузер с приложением
