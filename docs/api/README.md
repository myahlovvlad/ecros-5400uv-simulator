# CLI / API эмулятора

Симулятор поддерживает CLI-подобный интерфейс команд реального прибора. Он нужен для отладки и проверки логики без нажатий по web-панели.

## Базовые команды

| Команда | Назначение |
| --- | --- |
| `help` | Показать список команд |
| `connect` | Имитировать подключение |
| `quit` | Завершить сессию |
| `company` | Название производителя |
| `getsoftver` | Версия ПО |
| `gethardver` | Версия аппаратной части |

## Длина волны

| Команда | Назначение |
| --- | --- |
| `startwl` | Начальная длина волны |
| `endwl` | Конечная длина волны |
| `getwl` | Текущая длина волны |
| `swl 500` | Установить длину волны |
| `swm 500` | Имитировать поворот решётки |

## Измерения и калибровка

| Команда | Назначение |
| --- | --- |
| `rezero` | Калибровка нуля |
| `ge` | Измерение энергии |
| `ge 2` | Несколько измерений подряд |
| `adjustwl` | Калибровка длины волны |
| `diag` | Диагностика узлов |

## Усиление и параметры

| Команда | Назначение |
| --- | --- |
| `sa 1` | Установить усиление |
| `ga` | Получить усиление |
| `getslip` | Получить щель |
| `setslip 2` | Установить щель |
| `getsampler` | Получить sampler |

## Лампы

| Команда | Назначение |
| --- | --- |
| `setlampwl 340.5` | Установить λ лампы |
| `getlampwl` | Получить λ лампы |
| `wuon` / `wuoff` | Включить / выключить W-лампу |
| `d2on` / `d2off` | Включить / выключить D2-лампу |
| `getd2` | Состояние D2 |
| `getwu` | Состояние W |

## Темновой ток

| Команда | Назначение |
| --- | --- |
| `getdark` | Прочитать массив темнового тока |
| `resetdark` | Пересчитать массив темнового тока |

## Пример сессии

```text
> connect
ok.
> swl 500
> rezero
33869
1
> ge
33845
```

## Реализация

Основная обработка команд находится в:

- `src/application/services/CliService.js`

Команда изменяет то же состояние прибора, что и кнопки web-панели.
