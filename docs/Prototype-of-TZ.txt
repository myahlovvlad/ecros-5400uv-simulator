Питание: при падении напряжения ниже допустимого — немедленное сохранение критических данных и блокировка операций.

3. ГЛОБАЛЬНЫЕ ПРАВИЛА НАВИГАЦИИ И ПОВЕДЕНИЯ
ESC: возврат на предыдущий (родительский) экран. На экране ID: 1-5 (Главное меню) неактивна.

ID: 1-4 (Подождите...): отображается при выполнении длительных операций (перемещение монохроматора, калибровка, работа с файлами). Кнопки недоступны. Исчезает автоматически.

Навигация по спискам: ↑/↓ — перемещение курсора. ENTER — подтверждение.

Ввод значений: Keypad (цифры, точка, минус). CLEAR — удаление последнего символа. ↑/↓ — пошаговое изменение. ENTER — подтверждение.

Диалоги подтверждения (Да/Нет): ↑/↓ — выбор. ENTER — подтверждение. ESC — эквивалентен «Нет».

Активность функциональных кнопок (FILE, CLEAR, PRINT, SET, GOTOλ, ZERO, START/STOP): активны только на экранах, где это явно указано. На остальных экранах нажатие игнорируется.

7. АЛГОРИТМЫ ИЗМЕРЕНИЙ И КАЛИБРОВКИ
7.1. Измерение (START/STOP)
Выполняется 10 измерений с интервалом 100 мс.

Результат усредняется, отображается и сохраняется.

Если сигнал насыщен (A > 3.0), автоматически переключается усиление.

7.2. Калибровка по эталону (ZERO)
Выполняется команда rezero.

Возвращаемое значение энергии используется для установки усиления.

При успехе отображается экран ID: 1-6-1-GAIN.

7.3. Темновой ток (Dark)
Выполняется команда resetdark.

Сохраняется массив из 8 значений (для разных интегральных времён).

Применяется для коррекции всех последующих измерений.

7.4. Калибровка длины волны (Calib. WL)
Выполняется команда adjustwl.

Длительный процесс (до 30 с), во время которого показывается ID: 1-4.

По завершении — ID: DLG-OK или ID: DLG-ERR.

8.3. Пример сценария: Фотометрия
Вход в режим
getwl → отобразить текущую λ
ga → отобразить текущее усиление

GOTOλ
swl 500 → показать ID: 1-4, после ответа обновить экран

ZERO
rezero → дождаться <значение> и 1 → показать ID: 1-6-1-GAIN

START
ge 2 → получить два значения, усреднить, пересчитать в A/%T/E, добавить в таблицу

9. ОБРАБОТКА ОШИБОК И ВОССТАНОВЛЕНИЕ
9.1. Классификация ошибок
Код	Описание	Действие
E-01	Ошибка памяти	Блокировка сохранения, предложение удалить файлы
E-02	Нет сигнала лампы	Предложение заменить лампу
E-03	Сбой монохроматора	Предложение выполнить калибровку λ
E-04	Ошибка RTC	Запрос установки даты/времени
E-05	Ошибка USB	Повторная инициализация, сообщение
9.2. Сброс (Restore Default)
Сбрасываются все пользовательские настройки и удаляются все файлы.

Двойное подтверждение (диалог с предупреждением).

9.3. Watchdog
Сторожевой таймер с автоматической перезагрузкой при зависании (> 5 с без ответа).
