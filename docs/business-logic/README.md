# Бизнес-логика симулятора ЭКРОС-5400УФ

## Обзор

Симулятор воспроизводит работу спектрофотометра ЭКРОС-5400УФ — лабораторного прибора для измерения оптической плотности и пропускания образцов.

## Основные понятия

### Длина волны (Wavelength)

- **Диапазон:** 190–1100 нм
- **Точность:** 0.1 нм
- **Применение:** Выбор длины волны для измерения образца

### Оптическая плотность (Absorbance, A)

- **Диапазон измерений:** -0.3–3.999 Abs
- **Формула:** `A = -log₁₀(T)` где T — пропускание
- **Зависимость от образца:** Разные образцы имеют разные спектры поглощения

### Пропускание (Transmittance, %T)

- **Диапазон:** 0–200%
- **Формула:** `T = (I / I₀) × 100` где I — интенсивность после образца, I₀ — энергия 100%

### Энергия (Energy)

- **Диапазон:** 0–65000 условных единиц
- **Зависимость:** Нелинейная функция от длины волны с "рябью"

## Типы образцов

| Образец | Описание | Спектральные особенности |
|---------|----------|-------------------------|
| **reference** | Эталон / blank | A = 0 (полное пропускание) |
| **sampleA** | Образец A | Пики при 505 нм и 620 нм |
| **holmium** | Холмиевое стекло | 9 пиков: 241, 279, 287, 333, 361, 418, 451, 537, 640 нм |
| **kinetic** | Кинетический образец | Изменяется со временем (реакция) |
| **empty** | Пустая позиция | A = 3.2 (почти полное поглощение) |

## Физические модели

### 1. Справочная энергия

Функция `referenceEnergyAt(wavelength)` моделирует спектр источника света:

```
E(λ) = trend(λ) + ripple(λ)

trend(λ) = 33880 - |λ - 540| × 4.2
ripple(λ) = 320 × sin(λ/46) + 120 × cos(λ/23)
```

**Диапазон значений:** 12000–60000

### 2. Оптическая плотность образца A

#### SampleA (двухпиковый спектр):
```
A(λ) = 0.18 + 
       0.22 × exp(-(λ-505)² / (2×22²)) +
       0.11 × exp(-(λ-620)² / (2×35²))
```

#### Holmium (многопиковый спектр):
```
A(λ) = 0.04 + Σᵢ 0.9 × exp(-(λ-pᵢ)² / (2×2.2²))
где pᵢ ∈ {241, 279, 287, 333, 361, 418, 451, 537, 640}
```

#### Kinetic (временная зависимость):
```
A(λ,t) = baseline(λ) + rise(t)

baseline(λ) = 0.08 + 0.04 × exp(-(λ-546)² / (2×30²))
rise(t) = 0.75 × (1 - exp(-t/18))
```

### 3. Измерение образца

Функция `measureSample()` выполняет полный цикл измерения:

```javascript
1. Выбор значения темнового тока по усилению
2. Расчёт справочной энергии
3. Расчёт оптической плотности образца
4. Расчёт энергии после образца: E = E₀ × 10^(-A)
5. Добавление шума
6. Расчёт пропускания: T = (E - dark) / (E₁₀₀ - dark)
7. Расчёт оптической плотности: A = -log₁₀(T)
```

### 4. Темновой ток

Значения темнового тока для 8 уровней усиления:

| 2 | 74 |
| 3 | 152 |

## Режимы работы

### 1. Фотометрия

**Назначение:** Измерение оптической плотности/пропускания

**Операции:**
- Выбор длины волны (GOTOλ)
- Калибровка нуля (ZERO)
- Измерение (START/STOP)
- Сохранение результатов

**Данные:**
- Таблица измерений (до 100 записей)
- График зависимости A от номера измерения

### 2. Количественный анализ

**Назначение:** Определение концентрации веществ

**Методы:**
- **По коэффициенту:** C = K × A + B
- **По градуировке:** Построение калибровочной кривой

**Градуировка:**
- Число стандартов: 1–9
- Число параллелей: 1–9
- План измерений: N = standards × parallels

### 3. Кинетика

**Назначение:** Измерение скорости химических реакций

**Параметры:**
- Длительность: 10–1200 с
- Верхняя граница A: настраивается
- Нижняя граница A: настраивается
- Интервал измерения: 500 мс

**Данные:**
- Временная зависимость A(t)
- График кинетической кривой

### 4. Настройки

**Параметры:**
- D2-лампа (вкл/выкл)
- W-лампа (вкл/выкл)
- Темновой ток (калибровка)
- Калибровка длины волны
- Версия ПО/аппаратуры
- Сброс настроек

## Файловая система

### Группы файлов

| Группа | Расширение | Описание |
|--------|------------|----------|
| Фотометрия | .qua | Результаты фотометрии |
| Градуировка | .std | Калибровочные данные |
| Коэффициент | .cof | Коэффициенты K, B |
| Кинетика | .kin | Кинетические данные |

### Операции с файлами

1. **Открыть** — загрузка файла
2. **Переименовать** — изменение имени (макс. 18 символов)
3. **Удалить** — удаление файла
4. **Экспорт** — экспорт в CSV

### Валидация имён

```javascript
VALID_FILE_RE = /^[A-Za-zА-Яа-я0-9 _.-]+$/
```

**Правила:**
- Максимальная длина: 18 символов
- Разрешены: буквы (кириллица/латиница), цифры, пробел, _, ., -
- Пустое имя запрещено
- Дубликаты запрещены
