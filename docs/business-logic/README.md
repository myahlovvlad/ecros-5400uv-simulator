# Бизнес-логика симулятора

## Назначение

Слой бизнес-логики моделирует поведение спектрофотометра ЭКРОС-5400УФ без привязки к React и browser API.

## Основные сценарии

- расчёт эталонной энергии на длине волны
- расчёт оптической плотности для виртуальных образцов
- учёт темнового тока и усиления
- фотометрия
- кинетика
- градуировка
- файловые операции и preview USB-экспорта

## Виртуальные образцы

Поддерживаются несколько типов:

- `reference`
- `sampleA`
- `holmium`
- `kinetic`
- `empty`

Каждый тип задаёт свою модель зависимости поглощения от длины волны.

## Измерительная модель

Общий сценарий измерения:

1. рассчитывается эталонная энергия `referenceEnergyAt(wl)`
2. рассчитывается поглощение `absorbanceForSample(sample, wl, t)`
3. получается энергия после образца
4. добавляется шум
5. учитывается темновой ток
6. рассчитываются `%T` и `A`

## Градуировка

План градуировки строится по числу стандартов и параллелей. Каждый шаг имеет:

- код вида `С-1-1`
- индекс стандарта
- индекс параллели
- статус
- результат
