# Основные функции доменного слоя

## `referenceEnergyAt(wl)`

Возвращает модель эталонной энергии на заданной длине волны.

### Назначение

- базовая энергетика источника
- плавный тренд по диапазону
- небольшая рябь для правдоподобия

## `absorbanceForSample(sample, wl, t = 0)`

Возвращает оптическую плотность для выбранного виртуального образца.

### Особенности

- `reference` даёт нулевое поглощение
- `empty` моделирует почти полное поглощение
- `sampleA` содержит несколько полос
- `holmium` даёт узкие пики для проверки
- `kinetic` зависит от времени

## `measureSample({ sample, wavelength, gain, e100, darkValues, timeSec })`

Главная функция измерения.

### Возвращает

- `dark`
- `energy`
- `t`
- `a`

### Логика

1. выбирается темновой ток по усилению
2. считается эталонная энергия
3. считается поглощение образца
4. добавляется шум
5. считается пропускание
6. считается оптическая плотность

## `buildCalibrationPlan(standards, parallels)`

Строит линейный список шагов градуировки.

### Пример

Для `3 x 2` результат содержит:

- `С-1-1`
- `С-2-1`
- `С-3-1`
- `С-1-2`
- `С-2-2`
- `С-3-2`

## `buildUsbExportPreview(...)`

Готовит текстовый preview данных, которые будут экспортированы на USB.

Поддерживает группы:

- фотометрия
- градуировка
- коэффициент
- кинетика

## `initialDevice()`

Создаёт исходное состояние прибора со всеми параметрами по умолчанию.

## `validateFileName(name)`

Проверяет:

- непустое имя
- максимальную длину
- допустимый набор символов

## `validateWavelength(wl)`

Проверяет попадание длины волны в диапазон прибора.

## `validateNumeric(value, min, max, fieldName)`

Универсальная проверка числовых полей с ограничением диапазона.
