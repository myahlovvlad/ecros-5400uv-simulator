# ЭКРОС-5400УФ Симулятор

Симулятор спектрофотометра ЭКРОС-5400УФ — лабораторного прибора для измерения оптической плотности и пропускания образцов.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск разработки

```bash
npm run dev
```

Приложение откроется в браузере по адресу `http://localhost:3000`

### Сборка для продакшена

```bash
npm run build
```

### Запуск тестов

```bash
npm test
```

```

│   └── presentation/        # UI слой

```

## 🎮 Управление

### Клавиатура

| Клавиша | Действие |
|---------|----------|
| ↑ ↓ | Навигация по меню |
| Enter | Выбор |
| Escape | Назад / Отмена |
| 0-9 | Ввод цифр |
| . - | Ввод специальных символов |

### Кнопки панели

| Кнопка | Действие |
|--------|----------|
| ФАЙЛ | Файловый менеджер |
| ОЧИСТИТЬ | Удалить последнее измерение |
| ПАРАМЕТРЫ | Выбор величины (A/%T/Энергия) |
| ПЕРЕХОД λ | Установка длины волны |
| НОЛЬ | Калибровка ZERO |
| СТАРТ/СТОП | Измерение / Запуск кинетики |
| ВВОД | Подтверждение |

## 🔬 Режимы работы

### 1. Фотометрия
Измерение оптической плотности (A) и пропускания (%T)

### 2. Количественный анализ
- Построение градуировки
- Расчёт по коэффициенту (C = K × A + B)

### 3. Кинетика
Измерение скорости химических реакций

### 4. Настройки
- Управление лампами (D2, W)
- Калибровка темнового тока
- Калибровка длины волны

## 📊 CLI Интерфейс

Симулятор поддерживает эмуляцию CLI оригинального прибора:

```bash
# Подключение
connect → ok.

# Установка длины волны
swl 500

# Калибровка нуля
rezero → 33869

# Измерение энергии
ge → 33845

# Диагностика
diag → Filter=3, Lamp=3, ...
```

Полный список команд в [docs/api/README.md](docs/api/README.md)

## 🏗 Архитектура

Проект использует **Clean Architecture**:

```
┌─────────────────────────────────────┐
│     Presentation Layer (React)      │
│  - Компоненты                       │
│  - Хуки                             │
│  - Контексты                        │
├─────────────────────────────────────┤
│     Application Layer               │
│  - DeviceService                    │
│  - CliService                       │
│  - ScreenHandlers                   │
├─────────────────────────────────────┤
│     Domain Layer                    │
│  - measureSample()                  │
│  - validateWavelength()             │
│  - Constants                        │
└─────────────────────────────────────┘
```

См. [ARCHITECTURE.md](ARCHITECTURE.md) для подробного описания.
